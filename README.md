## Getting Started
```bash
yarn install
yarn start
```

## Usage
```javascript
<Container
	img="/img/card/pika.png"
	type="diagonal"
/>
```


## Reference
### Container Props
| Attribute | Type | Description |
|--|--|--|
| img | `string` | 이미지의 경로 ex) `/img/card/image.png` |
| type | `diagonal` `hologram-1` `hologram-2` | Card Overlay 스타일
